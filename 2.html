第一步：准备环境（只需做一次）

如果你是本地新建项目：

在终端

git remote add origin https://github.com/xiaofneg666/lian.git

git init

第二步：日常开发（不断循环）

1. 把所有改动放进暂存区（别忘了那个点！）

git add . 这个位置要注意空格和 .

2. 正式存入本地仓库（写好备注）

git commit -m "这次我写了什么功能"

# 3. 把本地进度同步到 GitHub

git push origin master

怎么回滚

可以先看看之前的历史纪录
